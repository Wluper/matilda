# Get Started Quickly

## Pre-requisites
1. Docker installed
2. Internet Connection

## Instructions for Local start
1. Build Docker Image of Matilda
```bash
# (in Matilda Root Folder)
cd web
docker build . -t matilda:beta
```

2. Start App
```bash
# (in Matilda Root Folder)
docker-compose up
```

3. Open App
- [Open Link in Browser](http://localhost:5000)
(localhost:5000)
